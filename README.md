# GetCryptoRate

A simple web wrapper around CoinGecko to fetch latest crypto currency rate

## Test Run
[Click to view](https://brave-brown-2d4de0.netlify.app/)

## Building

```bash
git clone https://github.com/xpt1x/GetCryptoRate.git
cd GetCryptoRate
npm install 
npm start
```
